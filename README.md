# 🛍️ Shop App

Modern e-commerce mobile UI built with React.

## 🚀 Features

- Onboarding Screen
- Login / Register صفحات
- Product Listing (Home)
- Product Detail Page
- Cart System (Add / Remove / Quantity)
- LocalStorage (cart сакталат)
- Responsive Mobile Design

## 🛠️ Tech Stack

- React
- React Router DOM
- Context API (State Management)
- CSS (Flexbox / Grid)
